# Paws & Whiskers Final Website

## Process
I started by creating my files and organizing everything as well as llinking it up to git. I got all the same images I used in a folder. I then made 3 seperate html files and one css file. I then just worked on one page at a time and then the css for it and adjusting the css as I went on!

## Challenges
The main challenges I faced were getting the containers to look how I wanted them to. As well as linking up all the html pages together. It was the first time I did it but after looking up how to do it, it was actually pretty simple. I also struggled with the css, it took a lot of trial and error to get everything looking similar to my wireframes. 

## What I Learned
Through this project, I learned how to build a multi page website from wireframes. I learned how to link mulitple html pages together and make a repsonisive design by using media queries!!
